Формулы расчёта суточной потребности

1. Формула Миффлина — Сан Жеора


let bmr = gender === "male"
  ? 10 * weight + 6.25 * height - 5 * age + 5
  : 10 * weight + 6.25 * height - 5 * age - 161;

Источник: Mifflin MD, St Jeor ST. Am J Clin Nutr. 1990

2. Расчёт суточной калорийности


const calories = bmr * activity;
const proteinTarget = (calories * 0.30) / 4;
const fatTarget = (calories * 0.30) / 9;
const carbTarget = (calories * 0.40) / 4;
